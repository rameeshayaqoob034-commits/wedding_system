# Wedding Hall Management System

## Project Overview
Wedding Hall Management System is a web-based application developed to manage wedding hall bookings and related services efficiently.

## Features
- User Registration & Login
- Hall Booking Management
- Customer Management
- Booking Details
- Admin Panel
- Responsive User Interface

## Technologies Used
- PHP
- HTML5
- CSS3
- JavaScript
- MySQL

## How to Run
1. Install XAMPP.
2. Copy the project folder to the `htdocs` directory.
3. Import the database (`.sql` file) into phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your browser using:
   http://localhost/YourProjectFolder/
